Graphic elements

SeMI uses distinct graphic elements to depict concepts inherent to the brand, its value proposition and products.

Principles â€“ How to construct graphic elements

The grid

In order to create a balance between consistency and flexibility SeMI uses an isometric grid as the starting point to generate graphic elements.

Networks

Lines and dots can be traced on the grid to draw networks.

SeMI uses a basic network pattern either as a standalone graphic element or as a component of illustrations.

Planes and volumes

Draw planes on the grid to generate volumes.

Rounded accents

To create contrast with sharp edges and accentuate certain elements, use rounded shapes.

Iconography

SeMI uses simple and straightforward iconography to represent concrete and straightforward concepts.

Illustrations

Use illustrations to grab the attention of an audience and to depict abstract concepts that are difficult to represent with a simple icon. Although illustrations remain mostly abstract, they should convey certain meaning.
