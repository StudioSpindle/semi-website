Beeswarm

The Beeswarm chart is a mix between the Bubble Chart and the Graph visualisations (Force-Directed Graph and Constraint-Based Layout). As opposed to the Bubble Chart, which is mainly used for aggregated data, the Beeswarm can be used to display the actual data. It arranges single nodes based on selected properties, allowing to get insights about the distribution of the data.

HTML / JS code snippet

<div id="beeswarm"></div> <script src="https://d3js.org/d3.v4.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.7.2/p5.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/p5.js/0.7.2/addons/p5.dom.min.js"></script> <script type="text/javascript" src="/js/charts.js"></script> <script type="text/javascript"> var file = '/assets/data/flavors_of_cacao.csv'; var options = { width: 650, height: 600, top: 0, left: 150, bottom: 110, right: 40, bubbleRadius: 3 }; beeswarm('beeswarm', file, options); </script>
