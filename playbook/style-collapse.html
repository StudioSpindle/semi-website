For collapsible elements the class "collapse--hidden" is used in conjunction with the content attribute hidden. The reason for this is that this proves to have the widest support in screen reader/browser combinations. As tested by developer.paciellogroup.com

The attribute data-toggle="collapse" defines that it's an collapse element.

Defining is done via the data-target="collapse". It will trigger an unique element, or multiple elements, with the corresponding class. In previous example "collapse".

Different types of triggers

The trigger should always be applied to a block level element, and the target should follow directly after.
