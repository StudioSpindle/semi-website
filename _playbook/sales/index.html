SeMI's Sales Playbook

SeMI's core solution is Weaviate which is publically available as an open-source project. At SeMI we sell upgrades of the open-source license (from startups to enterprise-grade), additional solution support, and we have a cloud offering. Together with our partners, we bring these solutions and services to the market.

Because Weaviate can be used for a wide variety of solutions, we have created an evergrowing set of sales plays that can be used to target potential customers.

In case you have questions about the sales plays or if you have any suggestions, make sure to let us know.
